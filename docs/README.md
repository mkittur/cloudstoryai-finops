
## Tech Stack
- OS: RHEL 9 (on-prem Dell server)
- Language: Python
- Database: PostgreSQL
- API: FastAPI
- AI: Local LLM (Ollama + Mistral, CPU-only)
- BI: PowerBI (via governed SQL views)
- DevOps: Git, SSH, VS Code Remote SSH

## Key Features
- Shared-service cost allocation
- Explainable anomaly detection
- Persona-based AI storytelling (CFO / FinOps / Engineer)
- Audio-based narrative output
- AI governance with ROI scoring
- Read-only BI access via SQL views

## Why This Matters
This project focuses on **how organizations actually operate**, not toy examples.
It demonstrates FinOps maturity, AI governance, and platform thinking.
